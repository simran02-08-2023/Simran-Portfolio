Project Overview

Desktop Banking Application

About

The Bank Management System is a Java desktop application designed to simulate real banking operations. It demonstrates GUI development using Java Swing, database connectivity with MySQL through JDBC, and object-oriented programming concepts.

Key Features

  • User Authentication
  • Account Creation
  • Cash Deposit
  • Cash Withdrawal
  • Balance Enquiry
  • Mini Statement
  • Fast Cash
  • PIN Change
